## Troubleshooting

### Common Issues

#### App Not Opening
1. **Restart device**: Power cycle your phone
2. **Clear cache**: Go to Settings > Apps > PathX > Storage > Clear Cache
3. **Reinstall**: Uninstall and reinstall the app

#### Notifications Not Working
1. **Check permissions**: Ensure notification permission is granted
2. **Battery optimization**: Disable battery optimization for PathX
3. **Do Not Disturb**: Check if Do Not Disturb is enabled

#### File Attachments Not Working
1. **Check file format**: Ensure file type is supported
2. **File size**: Large files may take time to load
3. **Storage space**: Ensure sufficient device storage

#### Data Not Saving
1. **Storage space**: Check available device storage
2. **App permissions**: Ensure storage permission is granted
3. **Restart app**: Close and reopen PathX

### Performance Issues

#### Slow Loading
1. **Clear cache**: Free up storage space
2. **Close other apps**: Free up device memory
3. **Restart device**: Clear system memory

#### UI Lag
1. **Reduce animations**: Disable system animations
2. **Lower screen resolution**: If available in settings
3. **Update device**: Ensure latest Android version
