## üé• **Technical Recording Guidelines**

### **Setup:**
1. **Screen Recording:** Use Android Studio's built-in emulator recording or ADB screen recording
2. **Audio:** Use clear, professional voiceover
3. **Resolution:** Record in high resolution (1080p or higher)
4. **Frame Rate:** 30fps minimum for smooth playback

### **Recording Tips:**
1. **Practice the script** before recording
2. **Speak clearly** and at a moderate pace
3. **Pause briefly** between sections
4. **Use consistent terminology** throughout
5. **Show actual data** rather than empty screens
6. **Demonstrate real scenarios** that users would encounter

### **Post-Production:**
1. **Edit out any mistakes** or pauses
2. **Add text overlays** for key points if needed
3. **Ensure audio quality** is clear
4. **Keep the video focused** on demonstrating criteria
5. **Add timestamps** for different sections if required
