# PATH X - Installation Guide

This guide will help you install and run the PATH X application on your Android device.

## Prerequisites

- Android device running Android 5.0 (Lollipop) or higher
- Minimum 100MB of free storage space
- Unknown Sources enabled (for APK installation)

## Installation Steps

### Step 1: Enable Unknown Sources

Before installing the APK, you need to allow installation from unknown sources:

**For Android 8.0 and above:**
1. Go to **Settings** > **Apps & notifications**
2. Tap **Advanced** > **Special app access**
3. Tap **Install unknown apps**
4. Select the browser or file manager you'll use to install the APK
5. Enable **Allow from this source**

**For Android 7.1 and below:**
1. Go to **Settings** > **Security**
2. Enable **Unknown sources**
3. Confirm by tapping **OK**

### Step 2: Download the APK

1. Download the `PATH_X.apk` file to your Android device
2. You can transfer it via:
   - USB cable from your computer
   - Email attachment
   - Cloud storage (Google Drive, Dropbox, etc.)
   - Direct download link

### Step 3: Install the APK

1. Open your device's **File Manager** or **Downloads** folder
2. Locate the `PATH_X.apk` file
3. Tap on the file to begin installation
4. Review the permissions requested by the app
5. Tap **Install**
6. Wait for the installation to complete
7. Tap **Open** to launch the app, or **Done** to finish

### Step 4: Launch PATH X

1. Find the PATH X icon in your app drawer
2. Tap to open the application
3. Follow any on-screen setup instructions

## Troubleshooting

### Installation Blocked
- **Issue**: "Installation blocked" message appears
- **Solution**: Make sure you've enabled installation from unknown sources (see Step 1)

### App Not Installing
- **Issue**: Installation fails or stops
- **Solution**: 
  - Check if you have enough storage space
  - Ensure the APK file downloaded completely
  - Try restarting your device

### App Crashes on Launch
- **Issue**: App closes immediately after opening
- **Solution**:
  - Check if your Android version is compatible
  - Clear app cache: Settings > Apps > PATH X > Storage > Clear Cache
  - Reinstall the application

### Permissions Issues
- **Issue**: App requests permissions but doesn't work properly
- **Solution**: Go to Settings > Apps > PATH X > Permissions and enable required permissions

## Updating PATH X

To update to a newer version:
1. Download the new APK file
2. Install it over the existing installation
3. Your data and settings should be preserved

## Uninstalling PATH X

To remove the app:
1. Go to **Settings** > **Apps**
2. Find and tap **PATH X**
3. Tap **Uninstall**
4. Confirm by tapping **OK**

## Support

If you encounter any issues not covered in this guide, please contact support or consult the official documentation.

## Security Note

Always download APK files from trusted sources to protect your device from malware. Verify the source of your PATH X APK before installation.
